# Waloi

Next.js -website 
## Teknologiat

- **Next.js**: 9.1.1
- **React**: 16.10.2
- **Node.js**: Yhteensopiva v22.0.0+ (legacy OpenSSL -tilan kanssa)

## Asennus

```bash
npm install
```

## Kehityspalvelimen käynnistäminen

```bash
npm run dev
```

Sovellus käynnistyy osoitteessa [http://localhost:3000](http://localhost:3000)

## Tuotantoversion buildaaminen

```bash
npm run build
npm start
```
